# Privacy Policy

Export Ulysses processes files locally on your Mac.

## Data Collection

The app does not collect, transmit, sell, or share personal data, usage data, diagnostics, note contents, or migration metadata. It contains no analytics, advertising, account system, or network service.

## File Access

The Mac App Store version accesses only the Ulysses backup and destination folders you select with macOS file panels. Security-scoped bookmarks may be stored locally in the app's sandbox so those choices can be restored on a later launch. You can revoke access by removing the app or resetting its container data.

The direct-download version can discover Ulysses' local backup folder when macOS permits access. The command-line tool reads only the backup path supplied or discovered and writes only to its requested destination and temporary staging sibling.

## Retention And Deletion

The app retains no copy of note content. Exported FSNotes files remain at the destination you selected until you delete them. Removing the app does not remove exports stored outside its sandbox.

## Support Reports

The optional anonymous support report excludes note contents, titles, filenames, URLs, and filesystem paths. Nothing is uploaded automatically; you decide whether to save or share it.

## Contact

Privacy or security concerns can be reported through the repository's private GitHub Security Advisory feature: <https://github.com/deverman/export-ulysses/security/advisories/new>.
